This page explains how to configure a custom domain for your vsForms instance.
## Custom Domain Setup

To use a custom domain (or subdomain)with vsForms:

1. Purchase a domain (if needed)
2. Configure DNS records to point to your vsForms instance
3. Update vsForms configuration:
   - Set `APP_URL` in `.env`
   - Set `NUXT_PUBLIC_APP_URL` and `NUXT_PUBLIC_API_BASE` in `client/.env`
4. Secure with SSL certificate
