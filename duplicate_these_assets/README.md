# VS Forms Assets Replacement Guide

This folder contains the original assets from OpnForm that need to be replaced with VS Forms branding. Use these files as templates for creating new VS Forms branded versions.

## Asset List and Replacement Instructions

1. **Logo Files**
   - `logo.svg` - Main logo used throughout the application (Path: `client/public/img/logo.svg`)
   - `logo.png` - PNG version of the logo (Path: `client/public/img/logo.png`)
   - Design the new logo using the VS Forms crimson and blue color scheme

2. **Favicon**
   - `favicon.ico` - Browser tab icon (Path: `client/public/favicon.ico`)
   - Create a small, simplified version of the VS Forms logo

3. **Social Preview Image**
   - `social-preview.jpg` - Image used when sharing links on social media (Path: `client/public/img/social-preview.jpg`)
   - Create a 1200x630px image featuring:
     - VS Forms logo
     - Tagline: "Professional Forms for Travel Advisors"
     - Travel-themed imagery
     - Crimson and steel blue brand colors

4. **UI Icons**
   - `subscription-modal-icon.svg` - Icon used in subscription modal (Path: `client/public/img/subscription-modal-icon.svg`)
   - `as-Select-dark.png` - UI element for dark mode (Update with VS Forms branding)

## Design Guidelines

- **Primary Color**: Crimson red (`oklch(55.66% 0.21 19.69deg)`)
- **Secondary Color**: Steel blue (`oklch(59.25% 0.09 239.65deg)`)
- **Typography**: Avenir/Montserrat font family
- **Brand Text**: "VS Forms by Voyager Social AI"
- **Target Audience**: Travel advisors and agencies

## After Redesign

1. Create new versions of these assets following the VS Forms brand guidelines
2. Replace each original file in its respective location in the codebase
3. Ensure file names remain the same to maintain compatibility with existing code
4. Test the application to verify all assets display correctly
